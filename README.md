# 安否リレー LP

`anpi.bantex.jp` へあとからCNAME移管するためのGitHub Pages静的LPです。

## 公開ルート

- GitHub Pages: `makoban/anpi-relay-pages`
- Current URL: `https://makoban.github.io/anpi-relay-pages/`
- Future custom domain: `anpi.bantex.jp`
- DNS予定: `anpi` CNAME -> `makoban.github.io`
- CNAME移管時は `CNAME.sample` を `CNAME` にリネームしてpushします。

## 画像

LP画像はCodex内蔵のgpt-image2生成画像を使用しています。
